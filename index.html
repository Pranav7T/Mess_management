Mess Management System

About This Project

This is a PHP-based web application for managing mess services. The system allows:

  • Users to find and subscribe to mess services
  • Mess owners to manage their services and menus
  • Admin to oversee the entire system

Local Installation

To run this project locally:

  1. Install XAMPP/WAMP/LAMP
  2. Clone the repository to your web server directory
  3. Import the database schema
  4. Configure the database connection
  5. Access via http://localhost/Mess_management/
